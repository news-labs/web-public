Official pricing for the Newsfork API. For the latest CTAs and trials, see [www.newsfork.com/pricing](https://www.newsfork.com/pricing/).

## Plans

| Plan | Price | Monthly requests | Highlights |
|------|-------|------------------|------------|
| **Free** | $0 | 1,000 | 10 countries, 24h freshness, keyword search |
| **Pro** | $79/mo | 50,000 | 100+ countries, semantic search + RAG, webhooks, News-V, MCP tools, 99.9% SLA |
| **Enterprise** | Custom | Unlimited | Dedicated workers, custom seeds, audit exports, up to 99.99% SLA |

## What counts as a request?

Any call to the Newsfork REST API — search, article fetch, feed query, semantic query, or MCP tool invocation — counts as one request. **Webhook deliveries do not count.**

## Free tier countries

The Free plan includes keyword search across **10 countries**. Pro and Enterprise include all **100+ countries**.

## Billing

- Pro includes a **14-day trial** (no credit card required to start).
- Payments are processed via **Stripe**.
- Enterprise customers can pay by **invoice**.

## Not yet available

The knowledge agent endpoint (`/v1/knowledge/ask`) is not yet generally available. Use semantic search via `/v1/rag/query` on Pro and Enterprise plans.
