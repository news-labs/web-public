## General inquiries

Email [hello@newsfork.com](mailto:hello@newsfork.com) for product questions, billing, and partnerships.

## Privacy and data requests

Email [privacy@newsfork.com](mailto:privacy@newsfork.com) for privacy-related questions or GDPR requests.

## API support

For integration help, see the [API documentation](https://docs.newsfork.com) or contact [hello@newsfork.com](mailto:hello@newsfork.com) with your account email and API key prefix (never send full API keys).

## Security reports

Report security vulnerabilities responsibly via [hello@newsfork.com](mailto:hello@newsfork.com) with subject line "Security Report".
