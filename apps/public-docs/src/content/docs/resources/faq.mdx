## Where are the API docs?

Technical API documentation lives at [docs.newsfork.com](https://docs.newsfork.com). Korean documentation: [docs.newsfork.com/ko/getting-started/](https://docs.newsfork.com/ko/getting-started/).

## What are the API pricing tiers?

See [Pricing](/company/pricing/) for current plan limits. Summary: **Free** ($0, 1K req/mo, 10 countries), **Pro** ($79/mo, 50K req/mo, all countries), **Enterprise** (custom).

## What counts as an API request?

Any REST or MCP tool call counts as one request. Webhook deliveries do not count toward your monthly quota.

## Is `/v1/knowledge/ask` available?

Not yet. Use `/v1/rag/query` for semantic search on Pro and Enterprise plans.

## How do I delete my account and data?

Email [privacy@newsfork.com](mailto:privacy@newsfork.com) from your registered account email. We process deletion requests within 30 days.

## Do you sell personal data?

No. See our [Privacy Policy](/legal/privacy/) for details on data sharing with infrastructure providers.

## Which law governs the Terms of Service?

The laws of the Republic of Korea. See [Terms of Service](/legal/terms/) section 8.

## Can I resell API access?

Not without a written partnership agreement. See [Acceptable Use Policy](/legal/acceptable-use/).
