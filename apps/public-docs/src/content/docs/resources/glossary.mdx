---
title: Glossary
description: Key terms used in Newsfork legal and policy documents.
lastUpdated: 2026-06-26
---

| Term | Definition |
| --- | --- |
| **API key** | Secret credential used to authenticate API requests |
| **Personal data** | Information that identifies or can identify a natural person |
| **Publisher** | Original source of news content indexed by Newsfork |
| **Quota** | Monthly request limit for your subscription plan |
| **Service** | The Newsfork API, website, and related products |
| **Subscriber** | Customer with an active Newsfork account or API key |
