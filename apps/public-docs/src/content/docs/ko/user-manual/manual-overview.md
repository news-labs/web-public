## 문서 구성

- **v1 API**: Seeds·Research·Health, Data Management, Queues 등 엔진 핵심 인터페이스.
- **v1 Guides**: 시스템 아키텍처, 배포·운영, 개발 워크플로우 (환경 설정, CI/CD, 로컬 테스트, 코드 품질, Cursor Rules).
- **User Manual**: 환경·배포·로컬 테스트, 번역 파이프라인, Logpush R2 설정.
- **Archive**: 프로젝트 마일스톤, 트러블슈팅 기록(Incident Reports), 경쟁사 비교, 변경 이력.
