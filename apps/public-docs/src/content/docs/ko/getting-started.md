## 소개

Newsfork Seeds는 뉴스 소스의 **발견(Research)** 과 **시드 계약(Seed)** 을 관리하는 시스템입니다.

- **Research**: 어디를 볼지(URL 발견)
- **Seed**: 어떻게 수집할지(콘텐츠 계약 정의)

Cloudflare Workers, Queues, R2, D1, KV를 사용하며, GitHub가 감사 추적(SSOT)을 담당합니다.
