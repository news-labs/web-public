## 소개

Newsfork Seeds는 뉴스 소스 **발견(Research)** 과 **수집 계약(Seed)** 을 REST API로 제공합니다.

- **Research**: URL 및 소스 발견
- **Seed**: 수집 방법 — 콘텐츠 계약 정의
